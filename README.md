# WaterMelonGame
Unity를 활용하여 개발한 2048 스타일의 2D 퍼즐 게임.

## 주요 기능
- 과일 합체 로직 (Physics 기반)
- 점수 저장 및 랭킹 시스템 (Firebase Realtime Database)
- 간단한 유저 행동 분석 (Firebase Analytics)
- 직관적인 UI 흐름 및 UX 개선

## 기술 스택
- Unity Engine (C#)
- Firebase (Realtime Database, Analytics)
